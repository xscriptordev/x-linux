<h2 align="center" id="how-to-contribute"> How to Contribute </h2>

<ol>
  <li><b>Fork</b> the repository to your own GitHub account.</li>
  <li><b>Create a branch</b> for your feature, theme or fix (<code>git checkout -b feat/amazing-theme</code>).</li>
  <li><b>Commit</b> your changes with clear and descriptive messages.</li>
  <li><b>Push</b> to your branch and open a <b>Pull Request</b> against the main branch.</li>
</ol>

<h2 align="center" id="project-structure"> Project Structure </h2>

<ul>
  <li><b>xscriptor-theme/</b>: The main theme pack for Android Studio, IntelliJ IDEA and other JetBrains IDEs.</li>
  <li><b>assets/</b>: Resources, icons and branding for the Xscriptor ecosystem.</li>
</ul>

<h2 align="center" id="rules"> Rules </h2>

<ul>
  <li>Always respect the <b>Code of Conduct</b> in all interactions.</li>
  <li>Do not include personal configuration files (like <code>.idea/</code> folders) or local IDE data in your PRs.</li>
  <li>For reporting security vulnerabilities, please refer directly to our <a href="SECURITY.md"><b>SECURITY.md</b></a>.</li>
</ul>
